## Installation

Install Ruby with RVM

`rvm install 2.5.1`

`rvm use 2.5.1 --default`

Install Bundler

`gem install bundler`

Now you can install gems and setup the DB

```
bundle install 
rails db:setup
```

Start your server

`rails s`
